# Maollar - B2B2C电商平台

## 项目简介

Maollar是一个基于Spring Boot和Vue.js开发的B2B2C多商户电商平台，支持PC、H5、小程序和APP多端应用。

## 功能特性

- 🛍️ **多端支持**: PC商城、H5商城、小程序、APP
- 🏪 **多商户**: 支持商家入驻，构建电商生态
- 🎯 **营销工具**: 优惠券、秒杀、拼团、砍价等
- 📊 **数据统计**: 订单统计、商品统计、流量分析
- 💳 **支付集成**: 支持微信支付、支付宝等
- 🚀 **高性能**: 基于微服务架构，支持分布式部署

## 技术栈

### 后端
- Spring Boot 2.7.x
- MySQL 8.x
- Redis
- Elasticsearch 7.3
- RocketMQ
- XXL-Job

### 前端
- Vue.js 2.x
- iView UI
- uni-app (移动端)

## 快速开始

### 环境要求
